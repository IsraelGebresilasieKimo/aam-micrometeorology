Doppler LiDAR VAD Wind & Turbulence Analysis

This page presents my Python-based workflow for processing and analyzing Doppler LiDAR observations using the Velocity–Azimuth Display (VAD) technique. The analysis focuses on retrieving wind profiles and turbulence-related quantities relevant to urban micrometeorology and Advanced Air Mobility (AAM).

his Colab notebook demonstrates how Doppler lidar VAD scans can be processed in Python to retrieve horizontal wind components, estimate vertical velocity, and derive turbulence statistics. The workflow uses xarray and NumPy to handle NetCDF data and is intended for boundary-layer meteorology, urban flow analysis, and CFD inflow validation. the data is obtained from https://www.cen.uni-hamburg.de/en/icdc/data/atmosphere/samd-st-datasets/samd-st-fesst-athome/fvah-tub-dlidcsm143-l1-any.html?utm_source=chatgpt.com and renamed as data.nc

The Jupyter notebook below is rendered using nbviewer to allow direct inspection of the code, methodology, and results. The workflow is fully reproducible and can be executed interactively in Google Colab.
