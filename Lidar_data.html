Doppler LiDAR Data Analysis

This page presents my Python-based workflow for analyzing Doppler LiDAR observations using the Velocity–Azimuth Display (VAD) technique. The notebook focuses on retrieving wind profiles and turbulence-related quantities from LiDAR measurements, with applications in urban micrometeorology and Advanced Air Mobility (AAM).

The analysis is implemented in Google Colab to ensure reproducibility and transparency. The full notebook is embedded below, allowing direct inspection of the code structure, processing steps, and outputs.
